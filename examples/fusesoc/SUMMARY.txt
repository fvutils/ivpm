EXAMPLE 1: Registry-Based Core Resolution (ex1-registry/)
========================================================
Demonstrates VLNV-based core resolution from the FuseSoC public registry.

Key Features:
  ✓ Resolves ::i2c:1.0 VLNV via fusesoc-cores registry
  ✓ Fetches FuseSoC from PyPI
  ✓ Includes Verilator simulator binary
  ✓ Auto-discovers .core files
  ✓ Generates FUSESOC_CORES environment

Quick Start:
  cd ex1-registry
  export PYTHONPATH=../../../src
  ivpm update
  . packages/packages.envrc
  fusesoc core list

EXAMPLE 2: Auto-Discovery of Cores (ex2-custom/)
=================================================
Demonstrates automatic core discovery in fetched packages.

Key Features:
  ✓ No explicit core configuration needed
  ✓ Fetches FuseSoC from PyPI
  ✓ Includes Verilator simulator binary
  ✓ Auto-discovers .core files in uart16550 core
  ✓ Easy to adapt for custom projects

Quick Start:
  cd ex2-custom
  export PYTHONPATH=../../../src
  ivpm update
  . packages/packages.envrc
  fusesoc core list
